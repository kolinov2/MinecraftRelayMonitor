# MinecraftRelayMonitor

A program thrown together out of curiosity to check online mode communication in Minecraft 26.2 Snapshot 7. It filters active network traffic for Microsoft Azure routing nodes on port 443, verifies them via the ip-api.com API, and displays their location along with a live ping graph.
